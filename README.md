# Simple-grade-student-tracker

# Description
This is a simple C++ program that allows users to input student grades and calculate basic statistics like average grade.

# Features
- Input multiple student grades
- Calculate average grade
- Simple and easy to use

# How to Run
1. Compile the program:
   g++ grade_tracker.cpp -o grade_tracker

2. Run the program:
   ./grade_tracker

# Technologies Used
- C++
- Standard Library (iostream, vector, etc.)
